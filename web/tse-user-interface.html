User Interface

As shown below, TSE is a single-window application segmented into regions. The largest region is the playfield, the grid where the user assembles circuits. On the left, from top-to-bottom, are the TS program, the JS snippet, and the output messages region.

The toolbar provides controls for TS program execution and element insertion:

  • Input - A user-entered binary string where each bit represents the state of an input node.
  • Run - Executes the TS program with the provided input node values. Run > Run and F6 do the same.
  • Components - Available components.
  • Add - Selects an available component. Click on a grid cell to insert it into the playfield.
  • Tetrominoes - Selects a tetromino in a particular orientation. Click on a grid cell to insert it into the playfield.

The status bar provides grid controls:

  • Center - Moves the origin to center-bottom by centering the horizontal scrollbar and bottoming the vertical scrollbar.
  • Depth - Recursive rendering depth, the level of displayed detail.
  • Width - Number of cells wide.
  • Height - Number of cells high.
  • Zoom - Grid cell size. Turn the scroll wheel to change it.
  • Cursor - Coordinates of the cell at which the mouse cursor points.
