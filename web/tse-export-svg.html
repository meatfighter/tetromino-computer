Export SVG

The Export SVG dialog, shown below, renders a circuit to an SVG file. It produced the diagrams for this text.

Select File > Export SVG... to open it.

Here are descriptions of its components:

  • Output
    • Stdout - When checked, the dialog exports to the standard output stream instead of a file.
    • File - When the stdout checkbox is unchecked, the dialog exports to a file with this name.
    • Browse... - Opens a file chooser for the export filename.
    • Default - Resets the filename to the default value.
  • Input
    • All possible values - When checked, the dialog renders the circuit for all possible input values to a single SVG. It works only when there are four or fewer input nodes.
    • Value - When All possible values is unchecked, the dialog uses this binary string to set the input node states.
  • Render
    • Absolute width - When checked, the dialog does not scale the SVG.
    • Display width - When Absolute width is unchecked, the dialog scales the SVG to this width.
    • Cell size - Unscaled grid cell size.
    • Depth - Recursive rendering depth.
    • Margin - Amount of pixel padding on all sides of the SVG.
    • Tetrominoes - Hides/shows the tetrominoes.
    • Structures - Hides/shows the dark rectangles that represent components.
    • Input nodes - Hides/shows the input nodes.
    • Output nodes - Hides/shows the output nodes.
    • Node values - Hides/shows the zeros and ones that indicate node states.
  • Grid
    • Visible - Hides/shows the grid.
    • Non-scaling stroke - When checked, grid lines are rendered at the same thickness regardless of Display width scaling.
    • y-axis - Hides/shows the y-axis.
    • Axes numbers - Hides/shows the axes numbers.
    • Open left - When checked, the grid appears open on the left, suggesting it continues beyond the displayed region.
    • Open right - When checked, the grid appears open on the right, suggesting it continues beyond the displayed region.
    • Open top - When checked, the grid appears open on the top, suggesting it continues beyond the displayed region.
    • Pad left - Number of empty columns to insert on the left.
    • Pad right - Number of empty columns to insert on the right.
    • Pad top - Number of empty rows to insert on the top.
  • Buttons
    • Export - Generates an SVG.
    • Close - Dismisses the dialog.

The animated tetromino diagrams were coded by hand. Some are modified versions of files produced by TSE.
